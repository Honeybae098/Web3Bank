## üîß LOCAL NETWORK SETUP

### Step 1: Start Local Hardhat Network
```bash
cd smart-contract
npx hardhat node
```

**Expected Output:**
```
Started HTTP and WebSocket JSON-RPC server at http://127.0.0.1:8545/
Local blockchain network ready for testing
```

### Step 2: Deploy Contract (if needed)
```bash
# In a new terminal, from smart-contract directory
npx hardhat run scripts/deploy.js --network localhost
```

**Expected Output:**
```
SmartBank deployed to: 0x9fE46736679d2D9a65F0992F2272dE9f3c7fa6e0
```

### Step 3: Configure MetaMask for Local Network

1. **Open MetaMask** and click on network dropdown
2. **Add Network** ‚Üí **Add Network manually**
3. **Network Details:**
   - Network Name: `Localhost 8545`
   - New RPC URL: `http://127.0.0.1:8545`
   - Chain ID: `31337`
   - Currency Symbol: `ETH`
   - Block Explorer URL: `http://localhost:8545` (optional)

4. **Switch to Local Network**

### Step 4: Fund Your Account
```bash
# Get test ETH from Hardhat accounts
# Check accounts in the node terminal:
# Account #0: 0xabc... (10000 ETH)
```

**Import Account to MetaMask:**
1. Copy private key from Hardhat node terminal
2. MetaMask ‚Üí Import Account ‚Üí Paste private key

## üõ†Ô∏è TROUBLESHOOTING

### Common Issues and Solutions

**Issue: "Contract not found" error**
- **Solution**: Ensure Hardhat node is running
- **Check**: Contract address in SmartBankConfig.js

**Issue: MetaMask not connecting**
- **Solution**: Switch MetaMask to Localhost 8545 network
- **Check**: Network ID should be 31337

**Issue: Transaction failing**
- **Solution**: Check account has sufficient ETH
- **Check**: Import Hardhat private key to MetaMask

**Issue: Balance not updating**
- **Solution**: Refresh browser after transaction
- **Check**: Wait for transaction confirmation

**Issue: Frontend not loading**
- **Solution**: Check if React dev server is running
- **Command**: `npm start` in Front-End directory
